About YAMADA SLOT

Overview

This is an online slot game platform made by django-framework and chart.js, ajax. It is made for studying personal and is currently under development. Currently, This project is the experimental project. Although it actually operates, there are many points to be improved.

Description of simulator

This is an online slot simulator which can see increase and decrease of medals by the graph. On the game screen, you can check the history of 100 games, and you can check the history of 2,000 games of each user on the admin screen. The explanation of each function is as follows.

  1. Game window
    2. You can check the actual game picture.
  2. Total Games
    3. After creating the account of user, count all the total games so far.
  3. Your Wallet
    4. Display getting number of total medals after creating the account of the user.
  4. Payout
    5. Since this is a simulator, The Player can choose of payout rates(95%, 100%, 105%).
  5. Game Number
    6. Display a game counts after depositing into slot game.
  6. Medal
    7. Display number of medals after depositing into slot game.
  7. Bet
    8. This button may be called "SPIN". Clicking on this button, three subtracted from the medal, and the game do one time.
  8. Deposit
    9. It is subtracted 50 medals from "your wallet" and deposited on 50 medals to "medal".
  9. Withdraw
    10. It will return the medal to "your wallet".
  10. Graph
    11. Display the history of past 100 games by graph.

This platform has a characteristic feature in the admin.

Turbo slot 1

Turbo slot2

If the function is checked, the probability of bell will rise significantly, The payout rate will be 140% The probability of bell fluctuates as follows.
